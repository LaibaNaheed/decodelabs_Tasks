# To-Do List Application

## Description

A simple Python command-line application that allows users to manage their daily tasks.

## Features

- Add new tasks
- View all tasks
- Menu-driven interface
- Beginner-friendly
- Uses Python Lists

## Technologies Used

- Python

## How to Run

```bash
python todo_list.py
```

1. Add Task
2. View Tasks
