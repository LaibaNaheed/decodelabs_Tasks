# Random Password Generator

## Description

This project generates a secure random password using Python.

## Features

- User chooses password length
- Includes uppercase letters
- Includes lowercase letters
- Includes numbers
- Includes special characters

## Technologies

- Python
- random module
- string module

## How to Run

1. Install Python.
2. Open terminal.
3. Run:

```
python password_generator.py
```

4. Enter password length.
5. Password will be generated.

## Example

Input:

12

Output:

A#8jL!4xQ@2p
