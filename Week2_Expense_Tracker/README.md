# Expense Tracker

## Description

A simple Python application that allows users to enter multiple expenses and calculates the total amount spent.

## Features

- Add multiple expenses
- Displays total expenses
- Shows expense list
- Calculates average expense
- Easy to use

## Technologies

- Python

## How to Run

```bash
python expense_tracker.py
```

## Example

Input

100
250
75

Output

Total Expenses: Rs. 425.00
